Analysis is based upon 5,519 properties identified using April 2012 GSA lease inventory. Multiple leases in the same building have been aggregated to calculate total rent and % leased, and remaining lease term and rent/RSF are presented using a weighted average. The data set excludes properties that are very small (<3,000 RSF) and properties for which no building size could be determined. Properties located in U.S. Territories (ex. Puerto Rico) are excluded from this analysis.
